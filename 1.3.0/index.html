MPAS-ToolsÔÉÅ

MPAS-Tools includes a python package, compiled Fortran, C and C++ tools, and scripts for supporting initialization, visualization and analysis of Model for Prediction Across Scales (MPAS) components. These tools are used by the Polaris and Compass packages used to create ocean, sea-ice, and land-ice test cases; the MPAS-Analysis package for analyzing simulations; and in other MPAS-related workflows.
